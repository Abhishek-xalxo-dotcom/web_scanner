# web_scanner

## Project Overview
web_scanner is a comprehensive web application security scanner designed to detect common vulnerabilities in web applications.

## Project Objective
To simulate automated web security testing and demonstrate understanding of common web vulnerabilities and scanning methodologies.

## Key Features
- Intelligent web crawling
- SQL Injection detection
- Cross-Site Scripting (XSS) detection
- Security header analysis
- Cookie security checks
- SQLite-based reporting dashboard

## Technologies Used
- Python
- Flask
- Requests
- BeautifulSoup
- SQLite

## Security Concepts Covered
- OWASP Top 10 vulnerabilities
- Input validation
- Automated vulnerability scanning
- Secure data handling
- Ethical security testing

## Project Workflow
1. Target URL is crawled within defined scope.
2. Input points are identified.
3. Automated vulnerability tests are performed.
4. Findings are stored and displayed via a dashboard.

## Disclaimer
This scanner is intended strictly for educational and authorized security testing purposes.
